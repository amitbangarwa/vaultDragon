# Vault Dragon

**Prerequisites :**

    * Node.Js
    * ES6 syntax

**Installation :**

    * Clone the Repo
    * cd your directory
    * `npm install`
    * `cd public`
    * `bower install`

**To start the local environment :**

    * `npm start`

T**o start production environment :**
    * change the config/production db url
    * `NODE_ENV=production npm start`

**To test the api :**

    * `npm test`
