# Vault Dragon

**Prerequisites :**

    * Node.Js
    * ES6 syntax
    * MongoDB

**Installation :**

    * Clone the Repo
    * cd vaultDragon
    * npm install
    * cd public
    * bower install

**To start the local environment :**

    * npm start

**To start production environment :**

    * change the config/production db url
    * NODE_ENV=production npm start

**To test the api :**

    * npm test
